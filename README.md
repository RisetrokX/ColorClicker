# ColorClicker
Clicks the color you set
## Installation
  1. Install dependencies
     `pip install -r requirements.txt`
  2. Clone this repo
     `git clone https://github.com/RisetrokX/Colorclicker.git`
  3. Change directory to it
     `cd ColorClicker`
  ### Linux
    sudo apt update
    sudo apt install gnome-screenshot
  ### Windows
   You don't have to do anything on Windows!
## Usage
  Run the script and enjoy ColorClicker:
   `python cc.py`
